Frontend

Инженеры фронтенда тесно сотрудничают с дизайнерами, чтобы сделать веб-сайты красивыми, функциональными и быстрыми.

Включает HTML, CSS, JavaScript, фронтенд, веб-приложения, React и многое другое.

Продвинутый JavaScript

JavaScript предоставляет множество возможностей для создания сложных и интерактивных веб-приложений. Продвинутые темы включают асинхронное программирование, работу с данными и взаимодействие с сервером.

Асинхронное программирование

Асинхронное программирование позволяет выполнять задачи, которые могут занять время (например, запросы к серверу), не блокируя основной поток выполнения.

Промисы

Объект, представляющий результат асинхронной операции. Промисы могут находиться в одном из трех состояний: ожидание (pending), выполнено (fulfilled) или отклонено (rejected).

Пример использования промиса:

let promise = new Promise((resolve, reject) => {
    let success = true;
    if (success) {
        resolve("Операция выполнена успешно!");
    } else {
        reject("Ошибка при выполнении операции.");
    }
});
        
promise
    .then(result => console.log(result))
    .catch(error => console.error(error));

Async/Await

Синтаксис, который упрощает работу с промисами. Ключевое слово async используется для объявления асинхронной функции, а await — для ожидания выполнения промиса.

Пример использования async/await:

async function fetchData() {
    try {
        let response = await fetch('https://api.example.com/data');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Ошибка:", error);
    }
}
        
fetchData();

Работа с таймерами

Таймеры позволяют выполнять код с задержкой или через регулярные интервалы времени.

setTimeout

Выполняет функцию один раз после заданного времени.

setTimeout(() => {
    console.log("Это сообщение выводится через 2 секунды.");
}, 2000);

setInterval

Выполняет функцию регулярно через указанные интервалы времени.

setInterval(() => {
    console.log("Это сообщение выводится каждые 2 секунды.");
}, 2000);

Работа с данными

JavaScript поддерживает работу с различными форматами данных, включая JSON.

JSON (JavaScript Object Notation)

Легкий формат обмена данными, который легко читается и пишется человеком, а также легко парсится и генерируется машинами.

Пример JSON:

{
    "name": "John",
    "age": 30,
    "city": "New York"
}

Работа с JSON в JavaScript

let jsonString = '{"name": "John", "age": 30, "city": "New York"}';
let jsonObject = JSON.parse(jsonString);
console.log(jsonObject.name); // John
        
let newJsonString = JSON.stringify(jsonObject);
console.log(newJsonString);

Введение в AJAX и Fetch API

AJAX (Asynchronous JavaScript and XML) позволяет обновлять части веб-страницы без перезагрузки. Fetch API — современный способ выполнения сетевых запросов.

Fetch API

Используется для выполнения HTTP-запросов. Возвращает промис, который разрешается в ответ.

Пример использования Fetch API:

fetch('https://api.example.com/data')
    .then(response => {
        if (!response.ok) {
            throw new Error('Network response was not ok');
        }
        return response.json();
    })
    .then(data => console.log(data))
    .catch(error => console.error('Ошибка:', error));
