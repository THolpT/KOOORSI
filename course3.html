Frontend

Инженеры фронтенда тесно сотрудничают с дизайнерами, чтобы сделать веб-сайты красивыми, функциональными и быстрыми.

Включает HTML, CSS, JavaScript, фронтенд, веб-приложения, React и многое другое.

Основы HTML

Что такое HTML?

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. Он используется для определения структуры и содержания веб-документа.

Структура HTML-документа

Основная структура HTML-документа включает в себя следующие элементы:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Заголовок страницы</title>
</head>
<body>
    <h1>Заголовок</h1>
    <p>Параграф текста.</p>
</body>
</html>

Данную структуру можно вызвать с помощью "Сниппета". Просто введите ! в VS Code и нажмите "Enter" или "Tab" на клавиатуре.

Описание элементов:

  • <!DOCTYPE html>: Декларация типа документа, указывающая, что это HTML5.
  • <html>: Корневой элемент HTML-документа.
  • <head>: Содержит метаинформацию о документе (кодировка, заголовок).
  • <body>: Содержит основное содержимое страницы.

Теги и элементы

HTML состоит из тегов, которые обрамляют содержимое и придают ему смысл.

Типы тегов:

  • Парные: Например, <p>...</p> для параграфа.
  • Одиночные: Например, <img> для изображений.

Парные теги всегда должны закрываться, иначе это может привести к ошибкам в коде.

Примеры HTML-тегов:

  • <h1>...</h1> — заголовки от первого до шестого уровня.
  • <p>...</p> — параграф текста.
  • <a href="https://example.com">Ссылка</a> — гиперссылка.
  • <img src="image.jpg" alt="Описание"> — изображение.

Атрибуты HTML

Атрибуты предоставляют дополнительную информацию об элементах. Они записываются внутри открывающего тега.

Примеры атрибутов:

  • href в теге <a> — указывает адрес ссылки.
  • src в теге <img> — указывает путь к изображению.
  • class и id — используются для стилизации и работы с JavaScript.

Семантические теги

Семантические теги помогают определить смысл элементов, улучшая доступность и SEO.

Примеры семантических тегов:

  • <header> — шапка страницы.
  • <footer> — подвал страницы.
  • <article> — независимый контент (например, блог-пост).
  • <section> — раздел документа.
  • <nav> — навигационное меню.

Формы и ввод данных

HTML-формы позволяют пользователям вводить данные и отправлять их на сервер.

Пример формы:

<form action="/submit" method="post">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name">
        
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Отправить">
</form>

Типы полей ввода:

  • <input type="text"> — текстовое поле.
  • <input type="email"> — поле для email.
  • <input type="password"> — поле для пароля.
  • <textarea>...</textarea> — многострочное текстовое поле.
  • <select>...</select> — выпадающий список.
  • <button>...</button> — кнопка.

Задание

Используйте полученные знания по тегам:

  • Откройте VS Code и создайте новый HTML-файл.
  • Используйте сниппеты для ускорения работы (например, введите p и нажмите "Enter" для быстрого создания тега <p>).
  • Попробуйте написать основные теги, не допуская ошибок.
  • Запомните смысл парных тегов и основной структуры HTML.
